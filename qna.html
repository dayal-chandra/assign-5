Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM like getElementById, getElementsByClassName, getElementsByTagName, querySelector and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Using innerHTML we can access the HTML code of the element, by innerText wew can access the text inside the HTML element and by the textContent we can also access the text inside the HTML tag.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript, using this technique we can add click event.

Question-4: What is event bubbling in the DOM?

The event bubble is a working process of event, it works from child element to the parent element.

Question-5: How do you create, add, and remove elements using JavaScript?

Using createElement, appendChild, and removeChild we can create, add and remove element in JavaScript.
